# 📊 Análise de Dados em Python (Versão Avançada)

Projeto desenvolvido por Yago Moreira com foco em análise de dados e visualização gráfica.

## 🚀 Funcionalidades
- Entrada de dados pelo usuário
- Cálculo de média, maior e menor valor
- Classificação automática dos dados
- Geração de gráfico com matplotlib

## 🛠️ Tecnologias utilizadas
- Python
- Matplotlib

## ▶️ Como executar
1. Instale a biblioteca necessária:
   pip install matplotlib

2. Execute o arquivo:
   python analise_dados_avancado.py

3. Insira os valores separados por vírgula (ex: 7,8,9,10)

## 🎯 Objetivo
Demonstrar habilidades iniciais em programação, análise de dados e visualização, com foco em evolução para projetos mais avançados em tecnologia e inteligência artificial.
